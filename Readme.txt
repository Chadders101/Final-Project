To use these files:

Download them both

1. Open your Google Extension by opening this link: chrome://extensions
2. click "Load Unpacked"
3. upload the "GoogleExtension" folder, found within this repo
4. Open a command prompt window and navigate to the directory you stored the Flask server
5. run the flask server command (install packages/libraries as neccassary)
6. open Yahoo Mail
7. Magic

This code also runs on a Cloud Flask server. The only two things which need changing is the location of the Machine Learning file, and the address of the server.
